Debugmode – Test mode, allows for more accurate testing. At the top of the screen (red text) the driven vehicle parameters are shown. In this mode, all failures are disabled (ie. derailing), CA / DSD (Czuwak Aktywny / Driver Safty Device, not acknowledging "SHP") does not cause emergency braking. In debugmode, opposite to the normal mode, certain keys have the following functions:

  • The F6 key – Speed up the clock time (Speed up the simulation's calculations, with Shift - 2x, Ctrl - 5x);
  • The Esc key – Speed up the simulation (might cause problems with launching certain events);
  • The F7 key – Show the mesh wire-frame / node triangles, does not work correctly;
  • The [ and ] keys – Shift the driven vehicle 100 meters to the front / back;
  • The q key – Stop the AI from driving your train further (after enabling it).

Turning on the AI (Shift + q) does not lock the keyboard. Also, the inside cab boundaries are gone.
This mode should normally NOT be used in the simulator.
